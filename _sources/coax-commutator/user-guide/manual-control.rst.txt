
*******************************************************
Manual Control
*******************************************************

.. TODO:: Add reset button

This guide instructs how to control the commutator using buttons on the commutator's front panel. This is useful for
situations such as when the user wants to quickly stop automated commutation or untwist the tether without disconnecting
it from the head stage.

.. NOTE:: These capacitive sensor buttons are not mechanical so they do not provide tactile or auditory feedback.

.. Important:: Manual controls override any ongoing remote controls. In other words, remote commands sent when a button is being pressed are ignored and cleared

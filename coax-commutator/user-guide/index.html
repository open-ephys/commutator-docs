User Guide#

Attention

If you haven’t verified the core functionality of your Open Ephys commutator, refer to the Quick Start Guide.

The coaxial commutator’s compatibility with lightweight high-bandwidth coaxial cables facilitate experiments by promoting natural animal behavior and enabling higher data transfer rates. This user guide details how to use the coaxial commutator.

How to Control Commutation with Bonsai and Key Presses#

This guide instructs how to perform commutation using Bonsai software and presses of the up/down arrows on the keyboard.

  1. Position the commutator such that gears can freely rotate

  2. Connect a computer to the commutator via the commutator’s micro-USB connector

  3. Download and configure Bonsai:

    1. Download and Install Bonsai if you haven’t already

    2. Open Bonsai and install the necessary package: Bonsai.StarterPack

    3. Download and open the .bonsai file (also known as workflow) for controlling the commutator using key presses

      commutator‑keypress‑control.bonsai

    4. Configure the PortName property of the SerialWriteString node to reflect the port to which the commutator is connected by left-clicking the SerialWriteString node and modifying the PortName property

  4. Run the workflow in Bonsai

    • Voila! If all above steps are correctly performed, the commutator motor rotates when the up and down arrow keys are pressed.

    • Angular displacement per key-press can be adjusted by left-clicking the String nodes and modifying the Value property when the workflow is not running
